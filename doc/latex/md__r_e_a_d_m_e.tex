\subsubsection*{About}

Simple 2D arcade style casual game. Chop wood, avoid branches, git gud. Written in C++/\+S\+F\+ML.

\subsubsection*{Dependencies}


\begin{DoxyItemize}
\item g++ or any other compiler with C++14 support
\item make
\item S\+F\+ML $>$= 2.\+5
\end{DoxyItemize}

\paragraph*{Linux}
